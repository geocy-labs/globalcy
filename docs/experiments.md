# Experiments

Milestone 1 supports one smoke path:

1. generate a Fermat bundle with GeoCYData
2. run `LocalPhiMLP` or `GlobalInvariantPhi`
3. inspect `metrics.json`, `summary.md`, and `predictions.parquet`

The first paper experiments will extend the same CLI flow to the Cefalu `lambda = 0.75` and `lambda = 1.0` cases once the symmetry-aware model is stressed on those regimes.
